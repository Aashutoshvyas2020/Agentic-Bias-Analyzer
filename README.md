# Agentic-Bias-Analyzer

A small project to evaluate bias in agentic systems. Contains frontend and backend components and evaluation scripts.

## Repository

This project is pushed to GitHub: https://github.com/Aashutoshvyas2020/Agentic-Bias-Analyzer

## Status ✅

- Public repository created and initial files pushed (2026-01-10).
- Latest commit: `e529a0e` — updated frontend and misc files.
- The `Evaluation/` folder and `.env` are intentionally ignored and not tracked.
- The existing `Backend` directory is an embedded repository and was left untouched.

## Structure

- Backend/: Node server and related files (kept as a separate git repo)
- Frontend/: Static site for demos (added/updated)
- Evaluation/: Evaluation scripts and caches (ignored in repo)

## Setup

- Install Node.js (for Backend)
- To run the backend server (if present):
  - cd `Backend` and run `npm install` then `node server.js` (or `npm start` if configured)
- The frontend is static; open `Frontend/index.html` in a browser for a demo.
- Create a `.env` file for secrets (ignored)

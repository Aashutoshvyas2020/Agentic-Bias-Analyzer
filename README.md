# Agentic-Bias-Analyzer

A small project to evaluate bias in news articles. Contains frontend and backend components and evaluation scripts.

## Structure

- Backend/: Node server and related files
- Frontend/: Static site for demos
- Evaluation/: Evaluation scripts and caches (ignored in repo)

## Setup

- Install Node.js (for Backend)
- Create a `.env` file for secrets (ignored)
